### Remove Subclasses

Este topico se refere a refatoracao Remove Subclasses, onde subclasses sem comportamento relevante sao removidas.
Quando as filhas so carregam pequenas variacoes de dados, a hierarquia vira custo sem ganho real de modelagem.
Ao remover subclasses desnecessarias, simplificamos o desenho e reduzimos cerimonia estrutural.
