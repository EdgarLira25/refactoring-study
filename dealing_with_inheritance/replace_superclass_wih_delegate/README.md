### Replace Superclass Wih Delegate

Este topico se refere a refatoracao Replace Superclass Wih Delegate, onde uma classe deixa de herdar de outra e passa a delegar comportamentos.
Quando uma subclasse herda apenas para reutilizar alguns metodos, a relacao de heranca pode nao representar o dominio corretamente.
Ao substituir por delegacao, reduzimos acoplamento hierarquico e ganhamos composicao mais explicita.
