### Replace Type Code With Subclasses

Este topico se refere a refatoracao Replace Type Code With Subclasses, onde um codigo de tipo e substituido por subclasses especificas.
Quando um objeto depende de condicionais por tipo para decidir comportamento, o codigo tende a crescer em ifs.
Ao trocar para subclasses, cada variacao encapsula sua regra e a evolucao fica mais previsivel.
