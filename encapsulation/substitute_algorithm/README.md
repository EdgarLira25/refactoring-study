### Substitute Algorithm

Este tópico se refere à refatoração Substitute Algorithm, onde um algoritmo existente é substituído por outro mais simples, legível ou eficiente, mantendo o mesmo comportamento esperado.
Quando uma rotina funciona, mas está complexa ou difícil de manter, trocar o algoritmo pode melhorar clareza e reduzir risco de erro em evoluções futuras.
A ideia central é preservar o resultado funcional enquanto melhora a forma de implementação.
