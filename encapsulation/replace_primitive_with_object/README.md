### Replace Primitive with Object

Este tópico se refere à refatoração Replace Primitive with Object, onde um valor primitivo que representa um conceito de domínio é substituído por um objeto.

Isso acontece em strings ou floats, isso pode ser melhor exemplificado com:
- Telefones como string
- Dinheiro como float
- Prioridade de tickets como string

Todos exemplos acima definem problemas que ocorrem quando definimos campos complexos como tipos primitivos que deveriam ter algum tipo de comportamento.
