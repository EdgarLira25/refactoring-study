### Encapsulate Record

Este tópico se refere à refatoração Encapsulate Record, onde uma estrutura de dados simples, como um hash map, é encapsulada dentro de uma classe.
Quando diferentes partes do sistema acessam diretamente os campos de um registro, o conhecimento da estrutura desses dados acaba ficando espalhado pelo código. Isso pode dificultar mudanças futuras e aumentar o acoplamento.
