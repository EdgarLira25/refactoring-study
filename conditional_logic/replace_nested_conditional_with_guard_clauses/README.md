### Replace Nested Conditional with Guard Clauses

Este topico se refere a refatoracao Replace Nested Conditional with Guard Clauses, onde estruturas aninhadas sao convertidas em clausulas de saida antecipada.
Quando `if/else` se acumulam em profundidade, o fluxo feliz fica escondido e aumenta o custo cognitivo para entender o comportamento.
Ao usar guard clauses, cada regra de excecao e tratada cedo e o caminho principal aparece de forma direta.
