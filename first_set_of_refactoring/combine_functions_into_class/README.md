### Combine Functions into Class

Este tópico se refere à refatoração **Combine Functions into Class**, onde várias funções que operam sobre os mesmos dados são agrupadas dentro de uma classe.
Quando funções separadas compartilham os mesmos dados ou representam partes de um mesmo comportamento, isso pode indicar que elas pertencem a um mesmo conceito do domínio.
Ao combiná-las em uma classe, o código passa a ter melhor organização, maior coesão e uma representação mais clara do comportamento associado aos dados.
