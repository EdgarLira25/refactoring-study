### Change Function Declaration

Este exemplo demonstra o padrão de refatoração Change Function Declaration.
Esse refactor consiste em alterar a assinatura de uma função para que ela represente melhor a intenção do código e as necessidades do domínio.
Durante esse processo é possível:
- Renomear funções para tornar o propósito mais claro
- Adicionar parâmetros quando a função precisa de mais informações
- Remover parâmetros que não são mais necessários
