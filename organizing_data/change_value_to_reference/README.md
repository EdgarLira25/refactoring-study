### Change Value to Reference

Este tópico se refere à refatoração Change Value to Reference, onde um dado tratado como valor passa a ser representado por referência compartilhada.
Esse movimento faz sentido quando identidade importa no domínio e diferentes partes do sistema devem apontar para o mesmo objeto conceitual.
Ao usar referência nesses casos, o modelo fica mais fiel ao negócio, evita duplicação de identidade e permite coordenar atualizações de forma consistente.
