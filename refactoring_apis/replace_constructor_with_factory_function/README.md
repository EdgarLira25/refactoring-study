### Replace Constructor With Factory Function

Este topico se refere a refatoracao Replace Constructor With Factory Function, onde a criacao de objetos e movida para funcoes nomeadas que explicam a intencao.
Quando o construtor precisa interpretar codigos ou combinar regras de criacao, ele vira um ponto opaco e dificil de evoluir.
Ao usar factory function, damos nome semantico para cada variacao e reduzimos logica de decisao no chamador.
