### Parameterize Function

Este topico se refere a refatoracao Parameterize Function, onde funcoes parecidas sao unificadas em uma so, recebendo os valores variaveis por parametro.
Quando temos varias funcoes quase iguais, a manutencao duplica e qualquer ajuste de regra precisa ser repetido em mais de um ponto.
Ao parametrizar a funcao, reduzimos duplicacao e deixamos a variacao de comportamento explicita nos argumentos.
