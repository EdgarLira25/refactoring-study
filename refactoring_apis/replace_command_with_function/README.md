### Replace Command With Function

Este topico se refere a refatoracao Replace Command With Function, onde removemos um command object desnecessario e mantemos apenas uma funcao simples.
Quando a classe comando nao possui estado relevante nem etapas complexas, ela adiciona cerimonia sem trazer beneficio real.
Ao trocar por funcao direta, reduzimos complexidade acidental e deixamos o fluxo mais objetivo.
