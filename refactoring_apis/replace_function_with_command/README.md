### Replace Function With Command

Este topico se refere a refatoracao Replace Function With Command, onde uma funcao complexa e transformada em um objeto comando com estado e etapas explicitas.
Quando a funcao acumula muitos passos e variaveis temporarias, fica dificil estender comportamento sem aumentar complexidade.
Ao mover para um command object, podemos organizar etapas, guardar contexto e facilitar evolucoes incrementais.
